# 🎯 Click Through Rate (CTR) Prediction Web App

## 📌 Overview

This project predicts whether a user will click on an advertisement using Machine Learning.

## 🚀 Features

* Predicts user click behavior
* Shows probability of click
* Web interface using Flask
* Real-time prediction

## 🧠 Technologies Used

* Python
* Machine Learning (Logistic Regression)
* Flask
* HTML & CSS
* Pandas, Scikit-learn

## 📊 Dataset

* Contains user data like age, income, internet usage, etc.

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/CTR-Prediction-App.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
python app.py
```

4. Open browser:

```
http://127.0.0.1:5000/
```

## 📈 Output

* Predicts: Click / No Click
* Shows probability %
